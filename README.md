# The Graveyard of Forgotten Technology
A spooky memory matching game built with Vite and Typescript. Includes a jumpscare which shuffles the graves
## Features
- Animations
- Graveyard generating
- Jumpscare & shuffling
## Usage
[Demo on Github Pages](https://hekinav.github.io/graveyard-game/)

### Running locally

1. `npm install` Install dependencies
2. `npm run dev` Run dev server (vite)

or

2. `npm run build` and `npm run preview` Build production files and run server
